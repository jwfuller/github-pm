GitHub Structure and Jargon

  • Users
  • Organizations
  • Repository (repo)
  • Issue
  • Project
Everyone signs into GitHub with a Personal Account. Personal accounts are then associated with our Organization Account. A Repository is a location in GitHub where files, code, and Issues are stored and managed. Repositories can be owned by Personal or Organization Accounts. Each Repository has option to enable the Issues feature which enables you to track ideas, tasks, bugs, or feedback. You can use Projects to organize and prioritize Issues.
